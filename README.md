# Resque::Benchmark

Benchmark your Resque jobs. Monitor how long they take from queuing to completion.

## Installation

Add this line to your application's Gemfile:

    gem 'resque-benchmark'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install resque-benchmark
